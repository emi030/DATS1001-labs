# DATS1001 - Data Science for All

Lab assignments from DATS1001 at George Washington University.

## Labs

### Lab 1 - R Basics
- Introduction to R syntax and data types
- Working with vectors, data frames, and basic operations

### Lab 2 - Data Visualization with ggplot2
- Bar charts, histograms, line charts, scatter plots
- Used `mtcars` and `ChickWeight` datasets
- Analyzed MPG by car model and chicken growth over time by diet

### Lab 3 - Data Wrangling with dplyr
- Filtering, grouping, and summarizing data
- Used V-Dem democracy dataset
- Analyzed democracy indicators and gender equality across countries (2000–present)

### Lab 4 - Advanced Analysis
- Statistical analysis and visualization

## Tools
- R / RStudio
- ggplot2, dplyr, tidyr
- Quarto
